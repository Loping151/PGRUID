## 说明

战双帕弥什插件

安装方式：

首先参考核心安装：https://docs.sayu-bot.com/

```bash
cd gsuid_core/gsuid_core/plugins
git clone https://github.com/Loping151/PGRUID
```

前置依赖：需要先安装 [XutheringWavesUID](https://github.com/Loping151/XutheringWavesUID)，本插件复用其请求底层、渲染引擎和数据库。

登录方式：

库洛账号通用，使用鸣潮插件登录后战双UID自动绑定。也可单独使用 `pgr登录` 进行登录。

催更/反馈/Bug/建议：与 [XWUID](https://github.com/Loping151/XutheringWavesUID) 首页的同一个群吧，不想再建群了。Issue亦会看。

## 丨其他

+ 本项目仅供学习使用，请勿用于商业用途。使用本插件视为同意提供用户凭据，用户凭据仅用于查询游戏数据。使用本插件造成的任何数据滥用行为与作者无关。
